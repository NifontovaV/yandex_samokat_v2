?# Тест на проверку получения заказа по треку заказа в Яндекс.Самокат с помощью API Яндекс.Прилавок.
- Для запуска тестов должны быть установлены пакеты pytest и requests
- Запуск всех тестов выполянется командой pytest
- Переменная URL_SERVIC в файле Configuration должна содержать URL запущенного тестового стенда Яндекс.Самокат
